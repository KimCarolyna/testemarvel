# Quiz Marvel 

## Descrição

Este é um quiz interativo que determina qual herói da Marvel combina com sua personalidade baseado em suas respostas a perguntas. Os heróis incluídos são Homem de Ferro, Capitão América e Viúva Negra.

## Instalação

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/quiz-marvel.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd quiz-marvel
   ```

## Como executar

1. Abra o arquivo `index.html` em um navegador web moderno (como Chrome, Firefox ou Edge).
2. Clique em "Começar Quiz" e responda às perguntas.
3. Ao final, veja qual herói combina com você.

## Estrutura do projeto

- `index.html`: Página principal do quiz, contendo a estrutura HTML.
- `script.js`: Arquivo JavaScript com a lógica do quiz, incluindo perguntas, pontuação e exibição de resultados.
- `style.css`: Arquivo CSS para estilizar a interface do quiz.
- `imagens/`: Pasta contendo as imagens dos heróis (homem_de_ferro.jpg, capitaoamerica.jpg, viuva123.jpg).

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Quiz com 10 perguntas.
- Sistema de pontuação baseado nas respostas.
- Exibição do resultado com nome, descrição e imagem do herói correspondente.
- Barra de progresso para acompanhar o andamento do quiz.
- Opção para refazer o quiz.

## Requisitos

Nenhum requisito especial. Basta um navegador web com suporte a JavaScript habilitado.
